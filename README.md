# JustDeco
 A simple mod for 7 days to die: Just deco!

# Purpose
This mod allows to craft some items from vanilla game by a normal player. Some of those items are sellable in traders and most of them can be created in Creative Mode but they are not being able to be crafted. So, this mod could help to those players who like to build inmersive construction.

It can be used as a server mod without problems because only adds some recipes and modify some blocks.

It is not purpose of the mod to impact in the game but help players in their decorations.

# Compatibility

Currently it is compatible with **game version 2.6**. Probably it can work with older versions but not guaranteed, it will be compatible if those versions got the items already because all of the items included in this mod come from the vanilla game.

# How to install the mod

- Download [latest version of the mod](https://github.com/carlospr/JustDeco/blob/main/dist/latest/JustDeco.zip). You can find other releases in this repository.
- In Steam, right-click on 7 Days to Die, choose Properties
- Click on the Local Files tab and click Browse Local Files
- Open the folder named Mods (create if if it does not exist)
- Unzip the file into that Mods folder. You will see a new folder with some files and a subfolder named Config.
